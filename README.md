# Smart Agriculture Monitoring System using C

This project simulates a smart agriculture monitoring system using C.
It monitors soil moisture, temperature, and humidity and decides irrigation status.

## Features
- Continuous monitoring
- Automatic irrigation decision
- Simple real-time simulation

## How to Run
gcc smart_agriculture.c -o agri
./agri

## Concepts Used
- while loop
- if-else
- Variables
- Real-time logic
